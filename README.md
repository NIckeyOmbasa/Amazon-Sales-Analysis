# Amazon Sales Performance & Strategic Dashboard

## 📌 Project Overview
This repository contains a comprehensive data analytics project modeled on an **Amazon India Junior Data Analyst** scenario. The goal of this mini-project is to clean transaction datasets, examine operational trends, build an interactive Excel dashboard, and perform multi-level statistical analysis using Python. 

The insights derived from this data assist executive management in identifying high-margin product lines, flagging underperforming territories, optimizing checkout pathways, and structuring corporate targets for upcoming operational quarters.

---

## 🛠️ Tech Stack & Skills Highlighted
- **Data Engineering & Manipulation:** Python, Pandas, NumPy
- **Visualizations & Reporting:** Matplotlib, Seaborn
- **Business Intelligence & Dashboard Design:** Microsoft Excel (Advanced Pivot Tables, Interactive Slicers, Contextual Dynamic KPI Blocks, Report Connections)
- **Methodologies:** Feature engineering (Calculated Columns), Retail Analytics, Descriptive Statistics
